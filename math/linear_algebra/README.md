# Linear Algebra

## Description

This project covers the fundamentals of linear algebra as applied to machine learning, implemented in Python using NumPy. Topics include vectors, matrices, transposition, dot products, matrix multiplication, and broadcasting.

## Learning Objectives

By the end of this project, you should be able to explain:

- What a vector is
- What a matrix is
- What a transpose is
- What the shape of a matrix is
- What an axis is
- What a slice is and how to slice a vector/matrix
- What element-wise operations are
- How to concatenate vectors/matrices
- What the dot product is
- What matrix multiplication is
- What NumPy is
- What parallelization is and why it is important
- What broadcasting is

## Requirements

- Python 3.5 (Ubuntu 16.04 LTS)
- NumPy 1.15
- pycodestyle 2.5
- No external module imports unless otherwise specified
- All files must be executable and end with a newline
- First line of every file: `#!/usr/bin/env python3`
- All modules, classes, and functions must have documentation
